ヒートアクションの演出にこだわりました。カメラの動きや敵のリアクションをプレイヤーが気持ちい良いと感じられるように調整しました。


//====================================================================================
// プログラム部分のアピールポイント
//====================================================================================
敵やアイテムの掴みの部分の制作に努力しました。一つのボタンで両方に対応するために、敵とアイテムの距離を比較し、最も近くのオブジェクトを掴むという処理にし、実現しました。


//====================================================================================
// 操作方法
//====================================================================================
[コントローラー]※XInputが対応したもの
・移動　左ステック
・視点　右ステック
・通常攻撃　Bボタン
・掴み　Yボタンでつかみ、Yボタンで離す
・回避　Aボタンでプレイヤーの向いている方向に回避
・ものを持って投げる　LBボタンでつかんで、掴んだ状態でLBボタンでその場に叩きつける　Aボタンで掴んだもので攻撃
・ヒートアクション　Xボタン


[キーボード、マウス]
・移動　ＷＡＳＤ
・視点　マウス
・通常攻撃　右クリック
・敵投げ　Ｅキーでつかみ、Eキーで離す
・回避　LSHIFTでプレイヤーの向いている方向に回避
・ものを持って投げる　Fキーでつかんで、掴んだ状態でFキーでその場にたたきつける　右クリックで掴んだもので攻撃
・ヒートアクション　Eキー
